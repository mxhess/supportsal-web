Getting Started with SupportSAL Pool

Everything you need to start mining Salvium on our pool

Step 1 - Install Wallet & Create Address ▼

Start here if you need a Salvium address and wallet:

  • Download the official Salvium Wallet from the official website
  • Create a new wallet and securely backup your seed phrase
  • Your Salvium address will start with SaLv and be 98 characters long
  • Keep your wallet synchronized to receive mining payments

Important: Never share your private keys or seed phrase with anyone!

Step 2 - Install Mining Software ▼

We recommend using XMRig for mining Salvium:

  • XMRig - Best performance for CPU and GPU mining
  • xmr-stak-cpu - Alternative CPU miner
  • Download from official sources only to avoid malware
  • Windows Defender may flag miners as false positives - add exclusions

XMRig supports both CPU and GPU mining with excellent optimization for RandomX algorithm.

Step 3 - Configure Settings ▼

Configure your miner with these pool details:

Pool Address:

pool.supportsal.com

Port Descriptions:

  • 3333 - Low-end hardware (50K difficulty)
  • 4444 - Mid-range hardware (250K difficulty)
  • 5555 - High-end hardware (800K difficulty)
  • 7777 - Cloud mining/NiceHash (2M difficulty)

XMRig Configuration Example:

{ "pools": [ { "algo": "rx/0", "coin": "salvium", "url": "pool.supportsal.com:4444", "user": "SaLv...(your wallet address)", "pass": "x", "rig-id": "worker1", "keepalive": true, "enabled": true } ] }

Command Line Example:

xmrig -o pool.supportsal.com:4444 -u SaLv...(your wallet address) -p x

Your Salvium Address: Enter your full Salvium wallet address as the username

Worker Name: Optionally add a worker name: youraddress.workername

Step 4 - Start Mining ▼

Launch your miner and start earning Salvium:

  • Run your configured miner software
  • Check that you're connecting successfully to the pool
  • Enter your Salvium address on this website to track your stats
  • It takes about 2 hours for your balance to start showing (blocks need confirmation)

Pool Details:

  • Pool Fee: 0.7%
  • Default Payout: 0.1 SAL minimum
  • Block Confirmation: 60 blocks
  • Payment System: RBPPS (Round Based Pay Per Share)
